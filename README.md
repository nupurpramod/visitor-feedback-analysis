# Visitor Feedback Analysis — Data Storytelling Project

A data storytelling project analyzing visitor feedback from a large exhibition — 
satisfaction ratings, recommendation rates, sentiment from open-ended comments, 
and patterns across zones, age groups, and discovery channels.

🔗 **Read the full story:** [Live blog post](https://nupurpramod.github.io/visitor-feedback-analysis/)

## About
During my internship, I worked on analyzing visitor feedback for a large 
exhibition. As the original dataset is confidential, this project uses a 
synthetic dataset built to reflect similar patterns and structure, while the 
analytical approach mirrors the real work.

## What's Inside
- `index.md` — the full data story (also published via GitHub Pages)
- `data/` — synthetic visitor feedback dataset (CSV)
- `notebooks/` — analysis script (Python: Pandas, NumPy)

## Tools Used
Python, Pandas, NumPy, Faker (for synthetic data generation)

## Key Insights
- Overall satisfaction: ~3.9/5, with 82% recommendation rate
- Sentiment from comments: ~70% positive, 25% neutral, 5% negative
- Busiest zones (Food & Culture, Kids Zone) had the lowest satisfaction — a 
  "victim of its own popularity" pattern
- Social media and word-of-mouth drove ~60% of awareness

*Note: All data is synthetically generated for demonstration purposes only.
